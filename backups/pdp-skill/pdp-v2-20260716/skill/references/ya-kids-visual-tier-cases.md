# YA Kids Visual Tier Cases

Use this file when diagnosing YA/Kids PDP visual tier cases across shoes, apparel, and accessories. It calibrates the reference logic from YA Kids visual research into the PDP scoring method.

This file is a case library, not a replacement for the 11-module scorecard. Do not change module weights or star bands based on this file.

## Source Logic To Calibrate

The YA Kids reference groups cases by category and visual tier:

| category | observed tiers |
|---|---|
| 鞋履 | T2 / T1 / T0 |
| 服饰 | T2 / T1 / T0 |
| 配件 | T2 / T2 / T2 in the provided board, with quality differences inside T2 |

Original reference definitions:

| original tier | source meaning |
|---|---|
| T2 | 只有白底图, 模特图或者平铺图 |
| T1 | T2 级别上增加外拍素材, 各类 assets |
| T0 | 有 VC 等资源, 按照 CDP 标准设计的 PDP |

Skill calibration:

| calibrated tier | scoring meaning |
|---|---|
| T2 | Basic material level. The page shows the product but has limited scene, proof, narrative, or visual attraction. |
| T1 | Standard conversion level. The page adds external shoot assets, on-body/on-foot, details, scenes, fit/size, and selling-point explanation. |
| T0 | Benchmark growth level. The page uses campaign/CDP/VC-level resources, but must also show story, visual system, proof translation, and brand memory. |

Important: source T0 does not automatically equal module `强`. It must still pass information quality, visual material quality, layout quality, category fit, and consumer-demand fit.

## YA Kids Case Positioning

YA/Kids pages usually sit between parent-led decision and child/teen desire.

Primary decision logic:

- Parent needs: safety, comfort, durability, fit, value, easy care, school/daily use.
- Kid/YA needs: color, trend, identity, sport scene, peer/social expression, fun, styling.
- Business needs: high SKU efficiency, seasonal update, campaign traffic, basket expansion, brand kids expertise.

When diagnosing, identify:

| dimension | examples |
|---|---|
| Category | shoes, apparel, accessories |
| Product role | basic SKU, seasonal core, new launch, hero SKU, campaign item |
| Decision owner | parent-led, kid-influenced, teen self-choice |
| Main risk | fit, comfort, safety, durability, styling, scenario, value |
| Visual target | T2 baseline, T1 conversion, T0 benchmark growth |

## Shoes

### T2 Shoes

Typical visible pattern:

- White-background or simple product images.
- Multiple colors and SKU thumbnails.
- Basic size table and product info.
- Limited on-foot, scene, or motion proof.
- Selling points are often listed but not visualized.

Use as a case for:

- Basic material baseline.
- Why product visibility is not enough for kids shoe decision.
- Medium scoring when modules exist but visual explanation is weak.

Module scoring implications:

| module | likely maturity | reason |
|---|---|---|
| 产品KV/封面故事 | 中 | Product is visible, but story and claim are weak |
| 沉浸式购物/场景化 | 弱 | No real scene; a generic white/gray studio model image alone cannot score |
| 卖点与功能证明 | 中 | Features may exist, but proof is text/listing-heavy |
| 细节查阅 | 中 | Angles/details exist, but benefit explanation is limited |
| 尺码/适配与对比选购 | 中/强 | Can be strong if size guidance is complete |

### T1 Shoes

Typical visible pattern:

- Adds external shoot assets, on-foot, kid movement, sport/daily scenes.
- Uses product details plus selling-point labels.
- More visual hierarchy than T2.
- Often includes size, scenario, color, and recommendation modules.

Use as a case for:

- Standard conversion upgrade from product listing.
- How external assets improve parent and kid decision efficiency.
- Medium-to-strong module scoring when information and visuals align.

Strong signs:

- The first screens clarify age stage, sport/daily scenario, and core selling points.
- Product benefits are mapped to shoe parts such as outsole, midsole, upper, heel, toe, or lining.
- On-foot and scene images help parents and kids imagine use.
- Color options are tied to styling, story, or scenario.

Medium signs:

- External shoot exists, but benefit proof is still weak.
- Scene photos are attractive but do not answer fit, comfort, grip, cushioning, or durability.
- Layout becomes long but not easier to decide.

### T0 Shoes

Typical visible pattern:

- Campaign or CDP-style visual system.
- Strong color, illustration, sport scenes, kid/YA lifestyle, and product hero moments.
- Selling points are more likely to be expressed through scene, motion, or graphic systems.

Use as a case for:

- Benchmark growth PDP for kids shoes.
- Visual memory plus decision proof.
- High standard for T0, not just "has campaign assets".

Strong signs:

- Product has a memorable hero story.
- Sport/daily scenes are tied to function.
- Visual system makes technology, comfort, durability, or styling easier to understand.
- Layout has rhythm, focus, and clear progression from desire to proof to decision.

Medium signs:

- Campaign assets are present but do not explain purchase concerns.
- Visuals are fun but lack size, fit, material, or functional proof.
- Page feels lively but not decision-efficient.

## Apparel

### T2 Apparel

Typical visible pattern:

- White-background, flat-lay, or basic model images.
- Front/back images, color blocks, size chart, product specifications.
- Limited styling or scenario.
- Fabric and fit may be described but not felt visually.

Use as a case for:

- Basic apparel information page.
- Medium scoring when fit/material modules exist but visual attraction is low.

Module scoring implications:

| module | likely maturity | reason |
|---|---|---|
| 产品KV/封面故事 | 中 | Product is shown, but style identity is weak |
| 沉浸式购物/场景化 | 弱/中 | Few lifestyle or school/sport scenes |
| 卖点与功能证明 | 中 | Fabric/function may be listed but not visualized |
| 细节查阅 | 中 | Detail images exist but do not fully explain comfort/craft |
| 尺码/适配与对比选购 | 中/强 | Can be strong if fit, model measurement, and size table are clear |

### T1 Apparel

Typical visible pattern:

- Adds model shoot, styling, kid movement, daily/school/sport scenes.
- Shows fabric, silhouette, color, outfit, and size/fitting.
- More brand and seasonal mood than T2.

Use as a case for:

- Standard conversion apparel PDP.
- How styling and on-body assets upgrade fit and desire.

Strong signs:

- Front/back/side fit and model measurements are clear.
- Fabric texture and wearing comfort are visible.
- Styling modules help parents and kids imagine daily use.
- Color options and outfit recommendations are connected.

Medium signs:

- Model images exist but pose and layout are repetitive.
- Styling is present but does not clarify silhouette, warmth/cooling, stretch, coverage, or care.
- Page has many images but low decision density.

### T0 Apparel

Typical visible pattern:

- Campaign/CDP-style narrative.
- Strong kid/YA lifestyle, sport, or street mood.
- Editorial color, typography, movement, and series storytelling.

Use as a case for:

- Brand-level kids apparel PDP.
- Turning product into identity, outfit, and seasonal story.

Strong signs:

- Product role and style identity are clear in the first screens.
- Outfit and series logic are visually memorable.
- Fabric/function proof is embedded into scenario and detail modules.
- Layout feels editorial but still helps purchase decisions.

Medium signs:

- Strong mood but weak fit/size/material proof.
- Big visuals but unclear conversion path.
- Series story exists but product-specific reasons are thin.

## Accessories

The provided board labels all accessory groups as T2, but visual quality varies. Treat this as an important calibration point: T2 can have internal levels, but it usually remains a basic material tier unless it adds scenario, proof, and layout narrative.

### T2 Accessories

Typical visible pattern:

- Product images and color options.
- Some model or scenario images.
- Simple feature blocks.
- For bags: capacity, compartments, ergonomic or school-use information may appear.
- For hats: headwear styling, sun/fit/adjustment may appear.
- For socks: color, graphic, comfort, and wearing context may appear.

Use as a case for:

- Basic accessory material baseline.
- Why "has cute visuals" or "has many product images" is not enough for T1/T0.

Module scoring implications:

| accessory type | key decision proof |
|---|---|
| Bags | capacity, compartments, carrying comfort, durability, school/sport scenario, body proportion |
| Hats | fit, head shape, brim, sun protection, breathability, adjustability, styling |
| Socks | length, thickness, cushioning, breathability, anti-slip, color/style, activity match |

Medium signs:

- Product is visible but proof is weak.
- Layout is product-listing-like.
- Cute or colorful visuals exist but do not answer use questions.
- Information exists but is not translated into visual decision guidance.

Strong signs for upgrading beyond T2:

- Bags show capacity, compartments, body scale, and school/sport scene.
- Hats show on-head fit, brim/closure details, sun or ventilation proof.
- Socks show cushion, pressure, anti-slip, breathability, length, and styling logic.
- Layout guides users from attraction to proof to choice.

## How To Use These Cases In Scoring

Use the YA Kids board as a calibration library:

1. Identify category: shoes, apparel, or accessories.
2. Identify source-like tier: T2, T1, or T0 based on material level.
3. Re-score with the PDP skill: module existence, information quality, visual material quality, layout quality, category fit, tier fit, and consumer-demand fit.
4. Do not let resource level override decision quality.
5. For unoptimized references, classify them as `T2 Basic Baseline`, `T1-minus Current Baseline`, or `T0 Positioning Gap` before assigning module maturity.

### Label Correction Rules

Use these correction rules when source labels conflict with scoring logic:

| source label | correction rule |
|---|---|
| T2 but has strong size/material proof | Can score specific modules `中/强`, but page visual tier remains T2 unless scene/layout/narrative improves |
| T1 but external shoot is generic | Score related modules `中`, not `强` |
| Unoptimized page with modules and weak visuals | Treat as `T1-minus Current Baseline`, not automatically T2 |
| Unoptimized page with only white-background/basic assets | Treat as `T2 Basic Baseline` |
| Campaign/hero product with T1/T2 execution | Treat as `T0 Positioning Gap` |
| T1 with strong decision proof and scan rhythm | Can score specific modules `强` under T1 conversion standard |
| T0 with campaign assets but weak proof | Score visual modules `中`; do not call it benchmark growth |
| T0 with campaign assets, proof translation, scene, and narrative | Can be used as `强` reference for T0 benchmark growth |
