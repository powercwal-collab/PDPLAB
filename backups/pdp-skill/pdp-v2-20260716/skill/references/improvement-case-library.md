# Improvement Case Library

Use this file when generating concrete improvement directions.

## P0 Cases

### Quick Decision Module

Use after a strong hero when buying reasons are not frontloaded.

Structure:

| reason | user benefit |
|---|---|
| 更轻 | 久走不拖脚 |
| 更软 | 落地更舒适 |
| 更稳 | 转向更安心 |
| 更百搭 | 通勤、轻运动、日常都能穿 |

### Suitable / Not Suitable

Use when the page does not clarify target user.

Example:

| user type | advice |
|---|---|
| 通勤久走 | 推荐 |
| 轻运动/健身 | 推荐 |
| 宽脚/高脚背 | 参考尺码反馈，必要时大半码 |
| 专业训练 | 不作为专业训练首选 |

### Proof Translation

Use when selling points are only numbers or slogans.

| selling point | proof | user benefit |
|---|---|---|
| 减重20% | 对比上一代重量 | 久走更轻 |
| 回弹提升50% | 中底材料测试 | 每一步更软弹 |
| TPU稳定结构 | 后跟/足弓结构图 | 转向更安心 |

### Parent Decision Frontload

Use for kids shoes when the parent-facing decision information exists but appears too late or is scattered.

Place these within the first two screens or first main-image sequence:

| decision item | example expression |
|---|---|
| 年龄段/尺码段 | 大童 32-40 欧码 / 幼童 14-22cm |
| 运动等级 | 入门 / 进阶 / 专业 |
| 适用场景 | 内场训练 / 外场竞赛 / 学校日常 |
| 适合打法或动作 | 速度型 / 力量型 / 跑跳转向 |
| 核心安心点 | 缓震稳定 / 防滑抓地 / 包覆锁定 / 耐磨外底 |

### Kids Main Image Funnel

Use when a PDP relies on plain product angles or scattered assets in the main image carousel. The main image carousel should act as a decision funnel, not a gallery.

Recommended sequence:

| frame | task |
|---:|---|
| 1 | 主图视频或封面: show product identity, age stage, and strongest claim |
| 2 | 年龄段&核心卖点: answer who it is for and why it matters |
| 3 | 卖点拆解: map benefits to shoe parts |
| 4 | 穿着场景: show sport context such as indoor training or outdoor competition |
| 5 | 穿着效果: show on-foot and full-body outfit effect |
| 6 | 多色呈现: show color options with nicknames or swatches |
| 7-8 | 细节卖点: show outsole, tongue, upper, toe, heel, lining, or other risk-reducing details |

If asset count is limited, prioritize frames 1-6 before extended detail frames.

## P1 Cases

### Scene + Benefit Binding

| scene | benefit | example copy |
|---|---|---|
| 通勤走路 | 轻量 | 从早到晚，脚步更轻 |
| 健身训练 | 稳定 | 深蹲、跳跃、转向更安心 |
| 城市休闲 | 百搭 | 运动鞋，也能穿出生活方式 |
| 轻跑热身 | 软弹 | 每一步落地更柔和 |

### Dynamic Demonstration

Use when a page claims easy wear, rebound, support, flexibility, or waterproofing.

Examples:

- Child self-wear video.
- Parent one-hand assist video.
- Midsole rebound animation.
- TPU support structure animation.
- Sole bending and anti-slip short clip.

### Detail Upgrade

Use when detail images exist but do not answer risk questions.

For kids shoes, show:

- Anti-slip outsole texture.
- Forefoot bending area.
- Toe bumper.
- Soft collar.
- Skin-friendly lining.

### Shoe-Part Function Breakdown

Use for kids performance shoes when claims such as cushioning, stability, traction, or durability are hard to believe from lifestyle images alone.

| shoe part | proof angle | parent benefit |
|---|---|---|
| 中底 | foam, rebound, compression, cushioning diagram | landing feels softer |
| 鞋面 | cables, stitching, overlays, wrapping structure | feet feel locked during turns |
| 外底 | traction pattern, outsole rubber, wear area | less slipping and better durability |
| 鞋舌/鞋口 | softness, padding, easy opening | easier wear and less rubbing |
| 后跟 | heel counter, stability structure | more confidence during quick stops |
| 鞋头 | bumper, reinforced material | toe protection during play |

### Color Nickname And On-foot Matrix

Use when color options are shown as plain swatches or product thumbnails.

Add:

- Memorable color nicknames.
- Product image plus on-foot image for each priority color.
- Styling or scenario cue for each color.
- Clear visual relation between selected color, model image, and purchase SKU.

## P2 Cases

### Stage-Based Recommendation

Use for kidswear, sports series, apparel series, or product families.

| stage or scenario | recommendation logic |
|---|---|
| 初学步 | 轻、软、防滑优先 |
| 稳步走 | 包裹和缓震更重要 |
| 爱跑跳 | 耐磨和支撑优先 |
| 幼儿园日常 | 好穿脱、好清洁、百搭 |

### Cross-age Size Entry

Use when kids products have toddler, preschool, grade-school, and big-kids versions.

Add:

- Current browsing version, such as "大童款".
- Adjacent buying entry, such as "选购幼童款".
- Size table across US, EU, China, and MM when relevant.
- Measurement method with foot length and width guidance.
- Growth and fit advice for high instep, wide feet, or between-size cases.

### Scenario Outfit Recommendation

Use when the PDP can expand basket size through sport or school scenarios.

| scenario | recommendation logic |
|---|---|
| 外场竞赛 | shoes plus breathable top, shorts, protective socks |
| 内场训练 | shoes plus training set, lightweight layer |
| 学校日常 | shoes plus easy-care outfit or backpack |
| 同系列升级 | entry/professional comparison by sport level |

### Page-End Closure

Use when the page ends without next action.

Add:

- Same series comparison.
- Color recommendation.
- Outfit or accessory bundle.
- Service and return reassurance.
- Brand or product proof recap.
