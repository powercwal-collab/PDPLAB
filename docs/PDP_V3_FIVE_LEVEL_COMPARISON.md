# PDP v3 五级成熟度规则与历史案例对比

更新时间：2026-07-16

## 结论

`pdp-v3` 将模块成熟度从 `弱 / 中 / 强` 扩展为 `弱 / 较弱 / 中 / 较强 / 强`，系数为 `0 / 0.25 / 0.5 / 0.75 / 1`。整体星级区间和 11 模块权重暂不改变，历史锁定版本不重算。

## 五级规则

| 成熟度 | 系数 | 服务端判定边界 |
|---|---:|---|
| 弱 | 0 | 无有效产品模块；标题、占位、通用模板、装饰素材和空壳不计分 |
| 较弱 | 0.25 | 有产品相关内容，但只有文案、单一信息、T2视觉或棚拍/模特图，图文未有效结合 |
| 中 | 0.5 | 产品信息与T2视觉基本结合，可回答基础购买问题，但证据或叙事不完整 |
| 较强 | 0.75 | 完整匹配信息与T1视觉有效结合，具备标准转化能力 |
| 强 | 1 | 完整可信信息、证据与T0视觉高度结合，形成品牌记忆与标杆增长表达 |

服务端会根据 `information_level`、`visual_tier`、`integration` 和受控证据类型执行上限，不接受模型直接绕过规则：

- 单一信息或视觉最高 `0.25`。
- T2 最高 `0.5`。
- T1 最高 `0.75`。
- `1` 必须同时达到 T0、完整可信信息与图文匹配。
- 产品相关 KV 文案或棚拍模特图不再直接归零，但最高只能为 `0.25`；通用内容仍为 `0`。

## 历史案例影子对比

本地数据库当前共有 10 个历史评分版本，全部绑定旧 `pdp-v1`，多数历史证据类型仍为未分型的 `page_region`。为避免把推测当成正式模型结果，本表采用保守影子规则：旧“弱/中”保持不变；旧“强”在缺少可验证 T0 字段时按 T1 重分类为“较强 0.75”。原版本不被修改。

| 实际项目（最新历史版本） | 旧分数 | 旧星级区间 | v3保守影子分 | v3星级区间 | 差值 |
|---|---:|---:|---:|---:|---:|
| Nike Kids｜毛毛虫幼童学步鞋 | 65.5 | 5星 | 55.75 | 4星 | -9.75 |
| Nike Kids｜儿童足球球衣 | 58.5 | 4.5星 | 52.25 | 4星 | -6.25 |
| GPT adapter smoke | 87 | 6.5星 | 66.5 | 5星 | -20.5 |
| 迪桑特 缓震流体跑鞋 | 79.5 | 5.5星 | 61.5 | 4.5星 | -18 |
| Nike Vomero 18｜缓震跑鞋详情页 | 89.5 | 6.5星 | 67.75 | 5星 | -21.75 |
| 球衣球员版 | 92 | 7星 | 69 | 5星 | -23 |

影子分下降不等于页面正式降分，主要说明旧三档规则把大量 T1 成熟模块直接计为满分。正式 `pdp-v3` 重跑时，模型必须重新输出信息完整度、视觉层级与结合度；真正达到 T0 的模块仍可保持满分，而产品相关的单一文案/棚拍模块可能由旧规则的 0 上升到 0.25。

## 典型差异

| 页面证据 | pdp-v2 | pdp-v3 |
|---|---|---|
| 通用标题、占位或模板 | 弱 0 | 弱 0 |
| 产品相关文案但无英雄视觉 | 弱 0 | 较弱 0.25 |
| 产品棚拍/模特图但无场景信息结合 | 弱 0 | 较弱 0.25 |
| 信息与T2视觉基本结合 | 中 0.5 | 中 0.5 |
| 完整信息与T1视觉结合 | 旧规则可能为强 1 | 较强 0.75 |
| 完整可信信息与T0视觉结合 | 强 1 | 强 1 |

## 正式对比验收条件

1. 对同一源文件新建 `pdp-v3` 诊断版本，不覆盖历史版本。
2. 确认 11 个模块均返回受控证据、信息完整度、视觉层级和结合度。
3. 对比总分、星级、模块系数、P0任务和“为什么还不是”说明。
4. 同一源文件至少重复运行三次，逐模块系数最大波动不得超过一个档位；若跨越两个档位则阻止正式锁定并标记模型稳定性异常。
5. 完成现有 Nike、迪桑特和球衣案例回归后，再决定是否调整整体星级区间。
